# Gaming DNS Generator

یک وب‌سایت ساده برای تولید آدرس‌های IP تصادفی جهت تنظیم به عنوان DNS در کنسول‌های بازی یا تنظیمات شبکه.

## ویژگی‌ها
- تولید DNS اولیه و ثانویه
- کپی آسان با یک کلیک
- تم تیره مخصوص گیمرها
- کاملا واکنش‌گرا (Responsive)

## تکنولوژی‌ها
- React 18
- Tailwind CSS
- Lucide Icons

## یادداشت‌ها
این پروژه صرفاً یک تولید کننده عدد تصادفی در فرمت IPv4 است و تضمینی برای عملکرد واقعی این آی‌پی‌ها به عنوان سرور DNS وجود ندارد.
